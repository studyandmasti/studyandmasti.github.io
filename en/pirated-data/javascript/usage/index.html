Usage

Read below examples for use this package in your project.

Generating first name

  • This method can generate random First name of a person.
  • returns firstName
const userData = require('pirated-data');
var startname = userData.firstNameGenerator();

Generating last name

  • This method can generate random Last name of a person.
  • returns LastName
const userData = require('pirated-data');
var lastName =userData.lastNameGenerator();

Generating Full name

  • This method can generate random Full name of a person.
  • returns FullName
const userData = require('pirated-data');
var fulName = userData.fullNameGenerator();

Generating Mobile Number

  • This method can generate random mobile number.
  • returns MobileNumber
  • Range of the mobile number in between(9999999999 - 6000000000)
const userData = require('pirated-data');
var mobnum = userData.mobileNumberGenerator();

Generating Financial card details

  • This function generate a virtual visa card Details
  • returns card holder name, card number, validThru, cvv
const userData = require('pirated-data');
var cardDetails = userData.financialCardGenerator();

output

Parameter Return Type Description
Full name String It is the full name of card holder
Card Number Number It is the financial card number
Valid thru String It is the expire date of card
CVV Number It is the required information of card like password 
["Haider Thapa", 4831558393843236, "10/2024", 309]

Generating AADHAR Number

  • This method can generate random AADHAR number of a person.
  • returns AADHAR Number
  • Range of the AADHAR number in between(999999999999 - 100000000000)
const userData = require('pirated-data');
var aadhar = userData.aadharNumberGenerator();

Generating Email-Id

  • This method can generate random Email-Id of a person.
  • returns Email
  • Domains supported now:
gmail yahoo hotmail rediffmail mns
github bandu baidu comcast aol 
const userData = require('pirated-data');
var eml = userData.emailIdGenerator();

Generating User Id

  • This method can generate random user Id of a person.
  • returns UserId
  • User Id is depend on the first name. It will Give an Alpha numeric UserId.
const userData = require('pirated-data');
var aadhar = userData.aadharNumberGenerator();

Generating Password

  • This method can generate random password for a user.
  • argument Length of Password
  • returns Password
  • It will Give an Alpha numeric password containing length of passing as argument.
const userData = require('pirated-data');
var pwd = userData.password(10);
Example:- AetY39GmQz
