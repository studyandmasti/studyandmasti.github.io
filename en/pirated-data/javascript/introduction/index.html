Introduction

Welcome to Pirated-Data!

Random Data generation library for the developer and test-Engineers.

Language supports

  • JavaScript

Coming up language support

  • Java
  • Kotlin

Features

  • Bunch data set
  • Single data Functions included

Getting Started

Install this package from the npm Library

npm i pirated-data

Adding package to Project

Befor Installing this package make sure you have installed node.js inside your machine. If already installed then open your command prompt inside the project root path and install by running above command mentioned.

Import package in Project

  • For java script project
const userData = require('pirated-data');
  • For Type-script project
import * as userData from pirated-data
