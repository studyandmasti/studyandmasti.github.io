Usage

Read below examples for use this package in your project (java)

Install to your project

  • If you are using maven project

    visit the website in search box type pirated-data

  • Add below mentioned snippet in to your pom file in dependency and save. It will be download to your project.

    • Always try to install latest available version to your project
<dependency>
  <groupId>io.github.studyandmasti</groupId>
  <artifactId>pirated-data</artifactId>
  <version>1.0.4</version>
</dependency>
  • Import the class from jar.
import io.github.studyandmasti.PiratedData;

PiratedData pd = new PiratedData();
