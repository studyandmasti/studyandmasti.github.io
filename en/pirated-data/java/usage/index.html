Usage

Read below examples for use this package in your project

Import the class from jar

import io.github.studyandmasti.PiratedData;

PiratedData pd = new PiratedData();

See uses

Generating random number

  • This method can generate random number
  • returns Number int
int num = pd.randomNumberGenerator(45);
System.out.println(num);

Generating hair color

  • This method can generate random hair color
  • returns hairColor String
`String` hairclr = pd.hairColorGenerator();
System.out.println(hairclr);

Generating birth date

  • This method can generate random date
  • returns birthDate String
String dob = pd.birthDateTimeGenerator();
System.out.println(dob);

  • This method can generate eye colour

Generating weight
