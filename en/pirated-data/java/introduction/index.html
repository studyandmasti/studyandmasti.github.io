Adding package to Project

Before Installing this package make sure you have installed Jdk inside your machine. Go to maven website and copy the desired code snippet in to your project, in corresponding file.

Import package in Project 🔌

Apache Maven 
<dependency>
  <groupId>io.github.studyandmasti</groupId>
  <artifactId>pirated-data</artifactId>
  <version>1.0.4</version>
</dependency>
Gradle Groovy DSL 
implementation 'io.github.studyandmasti:pirated-data:1.0.4'
Gradle Kotlin DSL 
implementation("io.github.studyandmasti:pirated-data:1.0.4")
